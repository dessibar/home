Project overarching goal

  • To ensure sustainable harvest of the capelin stocks through pooling research resources to develop common/shared tools for monitoring, stock projection and target rules for the capelin stock

Research goals

  • 1. Identify, compare and contrast key ecosystem traits that act as capelin dynamics drivers, and how these have changed over time
  • 2. Develop improved monitoring techniques and stock projection models
  • 3. Develop improved management decision support tools
  • 4. Develop end-user interface to the tools developed

Work packages

  • WP1: Establishment of dynamic database - Led by Gumundsdottir, Asta
  • WP2: Correlating ecosystem characteristics - Led by Gjøsæter, Harald
  • WP3: Tools for stock projection and advice - Led by Subbey, Samuel
  • WP4: Interface and dissemination tools - Led by Lamo, Yngve
